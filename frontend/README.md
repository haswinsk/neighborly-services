# Frontend (Neighborly Services)

This folder contains the React + TypeScript + Vite application.

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
npm run test
```

## Environment

Create `frontend/.env` from `frontend/.env.example`.

```bash
VITE_API_URL=http://localhost:5000/api
```

For full-stack orchestration, use root-level scripts from `../package.json`.
